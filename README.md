# EDA-Student-Performance-Analysis
Performed Exploratory Data Analysis (EDA) on a student performance dataset using Python, Pandas, and Matplotlib. Generated statistical summaries, correlation analysis, and visualizations to identify key factors affecting academic performance.

# Exploratory Data Analysis (EDA) - Student Performance Dataset
# Project Overview
This project performs Exploratory Data Analysis (EDA) on a student performance dataset to identify patterns, trends, and relationships among different academic factors.
The analysis includes:
* Dataset exploration
* Statistical summaries
* Missing value detection
* Correlation analysis
* Data visualization
* Insight generation

# Technologies Used
* Python
* Pandas
* Matplotlib

# Dataset Features
* Hours_Studied
* Attendance
* Assignments_Score
* Previous_Grade
* Final_Score

## Analysis Performed
# 1. Data Inspection
* Dataset shape
* Dataset information
* Statistical summary

# 2. Data Quality Check
* Missing value analysis
  
# 3. Correlation Analysis
A correlation matrix was generated to identify relationships between variables.

# 4. Visualizations
* Correlation Heatmap
* Hours Studied vs Final Score
* Attendance Distribution
* Final Score Distribution

# Key Findings
* Students who study more hours generally achieve higher scores.
* Attendance positively impacts final academic performance.
* Previous grades show strong influence on final results.
* Students with lower attendance tend to have lower final scores.
